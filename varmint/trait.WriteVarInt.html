A trait to allow writing integers to a byte-oriented sink in the varint format defined by Google's Protocol Buffer standard.

Required Methods

fn write_u64_varint(&mut self, val: u64) -> Result<()>

Write a u64 to this object encoded as a varint.

Errors

This function will return the first error returned by the underlying sink.

Examples

let mut bytes = vec![];
bytes.write_u64_varint(0x4B3FB5).unwrap();
